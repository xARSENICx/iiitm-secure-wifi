# Windows Wi-Fi Setup (Unofficial)

This community-maintained setup utility automatically configures the Wi-Fi profiles for IIITM.

## One-Click Setup

1. Download this entire repository as a ZIP and extract the `windows/` folder.
2. Right-click **`install.bat`** and select **Run as Administrator**.
3. Allow the Windows security prompt to proceed.
4. Once the setup completes, select **IIITM_Secure** or **IIITM_Secure_5G** from your Wi-Fi menu.
5. Enter your **IIITM LDAP Username** and **Password** when prompted.

## Troubleshooting

If the automated tool fails:

1. Ensure your Wi-Fi is turned on.
2. Try right-clicking **`install.bat`** and selecting **Run as Administrator**.
3. If you still cannot connect, please open an issue or contact **Ayush Sah** ([imt_2022026@iiitm.ac.in](mailto:imt_2022026@iiitm.ac.in)).
